Put static presentation slides here as full-slide assets.

Recommended formats:
- `PNG` for raster slides
- `PDF` for vector/static figure slides
- `JPEG` if the source is photographic

Then reference them from [presentation_deck.toml](../presentation_deck.toml) with:

```toml
[[slide]]
type = "image"
path = "assets/slides/your_slide.png"
```
